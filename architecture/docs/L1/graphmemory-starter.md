# spring-ai-ascend-graphmemory-starter — L1 Narrative

**Authority:** ADR-0147 (Structurizr Workspace Authority); narrative authored under W2 of the migration.

spring-ai-ascend-graphmemory-starter is the GraphMemory Spring Boot auto-configuration starter. Provides `GraphMemoryRepository` (tenant-scoped CRUD + semantic facts) and an in-memory reference implementation suitable for local dev.
